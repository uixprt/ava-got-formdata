# AVA GOT FormData

API integration test with [AVA](https://github.com/avajs/ava) and [GOT](https://github.com/sindresorhus/got) to make
file upload request.

## Install and Run

```sh
yarn && yarn test
```
* You need configure .env variables first to make it work in your API.
